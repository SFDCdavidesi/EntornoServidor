Realiza un programa en javascript que defina una caja de texto de forma que:
- Cuando pulse una tecla en esta caja de texto (onkeydown) el fondo de 
color de esta caja de texto se vuelva azul.
- Cuando pulse una tecla en esta caja de texto (onkeyup) el fondo de 
color de esta caja de texto se vuelva rojo.
